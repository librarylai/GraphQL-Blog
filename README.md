# GraphQL-Blog

## 啟動專案
> 安裝：yarn 

> 啟動：yarn dev 

> 產生 graphql generate 檔：yarn generate
## 使用套件
1. 前端/框架： Next.js、immer、use-immer、styled-components、graphql-codegen
2. 後端：GraphQL
3. DB： MongoDB
4. UI Library： Material-UI
